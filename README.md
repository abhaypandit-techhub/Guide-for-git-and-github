# Git & GitHub Setup Guide 🚀

This repository contains simple and practical notes for setting up Git and GitHub with VS Code. It covers repository creation, Git configuration, remote setup, and pushing code using both GUI options and terminal commands.

## 📚 What’s Included
- GitHub account and repository setup
- Git installation and configuration
- Connecting GitHub with VS Code
- Repository initialization
- Adding and managing remote repositories
- Git commands for add, commit, and push
- GUI and terminal-based workflow

## 💡 Why This Repository?
This guide is made for beginners who want to learn Git and GitHub in an easy and practical way while understanding the real workflow used in development.

## 🛠️ Tools Used
- Git
- GitHub
- Visual Studio Code (VS Code)
