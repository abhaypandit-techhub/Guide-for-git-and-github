                        *** CODE PUSH UP ON GITHUB ***

NOTES: If I understand correctly, you initialize a repository in VS Code and also add the remote URL and remote name, then you'll be able to push code.

METHOD - 1

-- First, stage all those files which you want to push on GitHub
-- Then, add a commit message
-- Then, click on the "tick indication arrow" just above the commit, wait a few seconds 0R Direct click on commit section add sum message then sinc to change again
-- Then, click the dot on the right side of the indication, go to the "push or pull" section, then click "push", wait a few seconds

    CONGRATULATIONS! YOUR CODE SUCCESSFULLY PUSHED ON GITHUB REPOSITORY

METHOD - 2
...Using terminal command

(i) git add . [Click Enter - means: all changes in source control files are staged and ready for commit]
(ii) git commit -m "put some message for commit"
NOTES: -m means adding a message to the commit

FINAL COMMAND
(iii) git push 'put remote name' 'branch name' [Click Enter - Branch may be common "main", this is also shown in VS Code which branch you have]

NOTES: USE THIS WHEN YOU WANT TO AVOID THE COMMAND "git add ." - then prefer:

(i) git commit -a -m "put some message for commit"
[Click Enter - -a means all files in change want to be staged and committed]

(ii) git push 'put remote name' 'branch name'
[Click Enter - Branch may be common "main", this is also shown in VS Code which branch you have]
