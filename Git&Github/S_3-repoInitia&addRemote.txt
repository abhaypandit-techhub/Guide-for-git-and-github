            *** REPOSITORY INITIALISE IN VS CODE AND REMOTE ADDITION ***

 ** Repository initialisation [ Method - 1 ]

** OPEN TERMINAL IN VS CODE TYPE TEXT ..
(i) git clone " provide url for repository " [Don't put double quote only url then click enter]

** Remote Addition [ Method - 1 ]

-- click on three dot choose ' remote '-- click on ' add remote '
-- then provide URL for 'HTTPS' for repository then click enter
-- then provide remote name ' origin ' [ "origin" general name choose as you wish]

NOTES : WHY WE NEED REMOTE ADDITION ?
-- for push our code in github [This is global repository]

** Remote Addition [ Method - 2 ]

.. USING COMMAND IN TERMINAL

-- git remote add <remote-name> <repository-url>
   [ press enter, don't include quotes; use spaces between the name and URL]

NOTES :

How to check remote ?
-- git remote -v
How to remove remote ?
-- git remote remove <remote-name> [only put the remote name; do not include quotes]

    

** Repository initialisation [Method - 2 ]

(i) Make a folder and open it in VS Code [if you already have a folder, you do not need a new one]
.. open VS Code terminal ..
(ii) git init  [Then press enter; after this the local repository is initialized]

** Remote Addition

-- click on three dot choose ' remote '-- click on ' add remote '
-- then provide URl for 'HTTPS' for repository then click enter
-- then provide remote name ' origin ' [ "origin" general name choose as you wish]

NOTES : If you put another repository link in the remote, then code push will go to that repository, not the initial local repository

***[M-3] Repository initialize *** Best way
i) Go to Source Control icon in VS Code
ii) Click 'Publish on GitHub'; after clicking, they ask for repository name, then click OK
